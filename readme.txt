名称：BrickPattern
+---------------------------------------------------------------------------+

技术支持：沐风课堂（http://www.bbvbb.com）

功能介绍：
BrickPattern for 3dsMax，是一款用MAXScript脚本语言开发的3dsMax小工具，可以创建2D可编辑样条线砖砌图形。可批量生成。生成的样条线可使用挤出修改器等转化为3D对象。
